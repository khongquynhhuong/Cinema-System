# HDQ Cinema

Full-stack hệ thống đặt vé xem phim: Spring Boot (API), React (UI), PostgreSQL, Docker Compose, Nginx (reverse proxy).

## Tech stack

- Backend: Java 21, Spring Boot 3  
- Frontend: React (Vite), npm  
- Database: PostgreSQL 17  
- DevOps: Docker, Docker Compose  

## Chạy dự án (sẽ bổ sung sau khi có đủ backend/frontend)

Tạm thời chỉ có PostgreSQL trong Docker Compose. Sau các commit tiếp theo sẽ thêm hướng dẫn đầy đủ.
